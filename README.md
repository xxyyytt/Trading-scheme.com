# Deriv Trading Dashboard

A simple trading dashboard connected to the Deriv API using WebSocket.

## Features

- Connect Deriv account
- Display account balance
- Show account ID
- Buy Rise (CALL)
- Buy Fall (PUT)
- Live API logs
- Real-time balance updates

## Technologies Used

- HTML
- CSS
- JavaScript
- Deriv WebSocket API
